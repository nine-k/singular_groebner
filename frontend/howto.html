Groebner calculator usage instructions

  • Field characteristic

    • 0 or prime number

  • Noncommutative algebra

    • If set to true, the non-commutative Grobner basis will be calculated

  • Variables

    • The list of variables. A variable can be a combination of letters and digits and underscores, but must begin with a letter. Newlines, commas and semicolons are valid separators

  • Ideal basis

    • Ideal expression. All numerical coefficients must be followed by a multiplication sign(*). Supports polynomial multiplication and the commutator operator [x, y] = xy - yx. If these operations are used, make sure that the max monomial degree value is set high enough. Newlines and semicolons are valid separators

  • Email

    • Email to receive computation results. Multiple emails can be passed, with a ; separator
